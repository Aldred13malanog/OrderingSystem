Project Overview:

This project is a simple Login System built in Java using `JOptionPane` for user interaction.
It demonstrates basic concepts of, arrays, and user input validation.

The system allows users to log in using a stored list of usernames and passwords.
It checks if the entered credentials match the saved data
and displays appropriate messages for:
* Successful login
* Incorrect password
* Non-existent account

How to Run:
1. Open the project in Apache NetBeans IDE.
2. Locate the file: `LoginSystem.java`.
3. Run the program (`Shift + F6` or click Run Project).
4. Follow the on-screen input dialogs.
